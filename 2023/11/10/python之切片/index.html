python之切片

发表于 2023-11-10

在 Python 中，切片（Slicing）是一种用于从序列（如列表、元组、字符串等）中获取子序列的方法。切片通过指定起始索引、终止索引和步长来定义。切片的基本语法如下：

其中：

  • start 是切片开始的索引，默认为序列的开头。
  • stop 是切片结束的索引（不包括该索引对应的元素），默认为序列的末尾。
  • step 是切片的步长，表示取值的间隔，默认为 1。
